# Flappy-Bird

## How to install?  
Before these steps make sure you have "Flutter 2.0"

*git clone https://github.com/rutuja-khaire35/portfolio-app.git  
cd portfolio-app  
flutter packages get  
flutter config --enable-web  
flutter run -d chrome*
